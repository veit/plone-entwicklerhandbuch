=========
Formulare
=========

Bisher haben wir nur Formulare erstellt, deren Inhalte von Archetypes in der ZODB gespeichert wurden. In diesem Abschnitt werden wir nun eigenständige Formulare programmatisch erstellen.

.. note::
    Mit `PloneFormGen <https://pypi.python.org/pypi/Products.PloneFormGen/1.5.5>`_ gibt es ein Produkt, das die einfache Erstellung von Formularen über ein Web-Interface erlaubt. PloneFormGen und die erforderlichen Zusatzprodukte lassen sich einfach installieren indem unter ``eggs`` im ``[buildout]``-Abschnitt ``Products.PloneFormGen`` eingetragen wird.
